🏊 Calculador de Volumen de Piscinas (Python)

Este es un programa sencillo en Python diseñado para calcular cuántos litros de agua se necesitan para llenar una piscina basándose en sus medidas.

🛠️ ¿Qué hace el programa?

- Pide las medidas: Solicita al usuario el largo, el ancho y la profundidad promedio de la piscina en metros.

- Calcula el volumen: Aplica la fórmula matemática para obtener los metros cúbicos

- Conversión a litros: Transforma automáticamente los metros cúbicos a litros

- Resultado claro: Muestra por pantalla la cantidad total de litros necesaria para el llenado.
